  • CSS:

    Cascading Style Sheet

    是一种以.css结尾的应用于html变现样式的文本文件

    样式性文本

    三种应用形式：

    • 行内样式： <tagName style="property:value;property:value;......" >tagName>
    • 内联样式：

      <style type="text/css">
      /*样式规则*/
      style>

    • 外联样式：
      <link type="text/css" href="css/*.css" rel="stylesheet" />

    元素选择符：

    • 通配选择符：
      *{
      property:value;
      property:value;
      ......
      }
    • id选择符：
      #id{
      property:value;
      property:value;
      ......
      }
    • 类选择符：
      .className{
      property:value;
      property:value;
      ......
      }
    • 标签选择符：
      tagName{
      property:value;
      property:value;
      ......
      }
    • 关系选择符：
      • 包含选择符：
        selector selector{
        property:value;
        property:value;
        ......
        }
      • 父子选择符：
        selector>selector{
        property:value;
        property:value;
        ......
        }
      • 兄弟选择符：
        selector~selector{
        property:value;
        property:value;
        ......
        }
      • 相邻选择符：
        selector+selector{
        property:value;
        property:value;
        ......
        }
    • 属性选择符：
      • 包含属性选择符：
        selector[property]{
        property:value;
        property:value;
        ......
        }
      • 等于属性值选择符：
        selector[property="value"]{
        property:value;
        property:value;
        ......
        }
      • 包含属性值选择符：
        selector[property*="value"]{
        property:value;
        property:value;
        ......
        }
      • 包含单词属性值选择符：(空格分隔)
        selector[property~="value"]{
        property:value;
        property:value;
        ......
        }
      • 包含以单词属性值开头选择符：(减号分隔)
        selector[property|="value"]{
        property:value;
        property:value;
        ......
        }
      • 包含以属性值开头选择符：
        selector[property^="value"]{
        property:value;
        property:value;
        ......
        }
      • 包含以属性值结尾选择符：
        selector[property$="value"]{
        property:value;
        property:value;
        ......
        }
    • 伪类选择符：
      a:link,a:hover,a:active,a:visited{
      property:value;
      property:value;
      ......
      }
      selector:hover{
      property:value;
      property:value;
      ......
      }
    • 伪对象选择符：
      selector:before{
      content:value;
      property:value;
      ......
      }
      selector:after{
      content:value;
      property:value;
      ......
      }

  • JavaScript:

    是一种以.js结尾的用于html中的一个文本文件

    行为式文本

      使用方式：
    • 行内行为： <tagName action="js操作" > tagName>
    • 内联行为：
      <script>
      js code;
      script>
    • 外联行为：
      <script type="text/javascript" src="*.js">script>
    • action: onclick,onmouseover,onmouseout,onchange,onfocus,....
      JS的组成：
    • ECMAScript：编程语言的基本语法
      • 变量
      • 运算符
      • 数据类型
      • 控制语句
      • 函数
      • 数组
      • 定义变量：
        var varName = value;
    • DOM：Document Object Model
      • 操作：
        • 查找元素：(返回结果，元素对象)
          document.getElementById('id');
          document.getElementsByTagName('tagName');
          document.getElementsByClassName('className'); //非IE浏览器
        • 添加元素：
          parentElement.appendChild();
        • 修改元素：
          element.property = value;
          element.style.property = value;
          element.innerHTML = value;
        • 删除元素：
          parentElement.removeChild();
      • div,找到这个div元素,设置其innerHTML=value,更改其属性
      • 定时器：
        • setInterval();
        • clearInterval();
      • 超时器：
        • setTimeout();
        • clearTimeout();
    • BOM：Browser Object Model
